# Prisma Demo

Prisma Integration Demo by Sahan Sachintha using typescript and nodejs

## Getting Started

First, run the development server:

```bash
pnpm i
# or
npm i
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
